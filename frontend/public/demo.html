How to embed on a real site

This same widget works on any website. Paste this just before </body>:

<script src="https://web-talk-ai.vercel.app/widget.js?v=2.1.4"></script>
<script>
  WebTalkAI.init({
    apiKey: "wtk_YOUR_KEY",
    position: "bottom-right",
    theme: "purple",
    voiceEnabled: true,
    ttsAutoPlay: true
  });
</script>
